  | ---------------- | ------------------------------------------------------------ | ------------------------------------- |
  | /user/login      | POST                                                         |                                       |
  | 参数             | 描述                                                         | 必须                                  |
  | mobile           | 手机号                                                       | 是                                    |
  | password         | 密码                                                         | 是                                    |
